# 📊 Courier Logistics Performance Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze courier shipment performance, revenue trends, and operational efficiency.

The dashboard provides clear, data-driven insights to support decision-making and improve business performance.

---

## 🎯 Objectives
- Monitor total revenue and shipment volume
- Track failed shipments and revenue loss
- Analyze shipment distribution by province
- Evaluate service type performance
- Identify trends over time

---

## 📊 Dashboard Features

### ✅ KPI Cards
- Total Revenue
- Total Shipments
- Failed Shipments
- Failed Revenue

### 📈 Visualizations
- Shipments by Province (Clustered Bar Chart)
- Revenue Trend Over Time (Line Chart)
- Shipments by Service Type (Column Chart)

### 🎛 Interactivity
- Province slicer
- Service Type slicer

---

## 🗄 SQL Analysis
SQL queries were used to extract, aggregate, and validate data before visualization in Power BI.

Key queries include:
- Revenue aggregation
- Shipment distribution by province
- Monthly revenue trends
- Service type performance

---

## 🛠 Tools & Technologies
- Power BI
- SQL (SQL Server)
- DAX (Data Analysis Expressions)
- Data Modeling

---

## 📊 Data Source
This project uses a simulated dataset created for learning and portfolio purposes. The dataset represents courier logistics operations, including shipments, revenue, and service types.
